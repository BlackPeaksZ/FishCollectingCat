<h3>Простая игра "Кот собиратель рыбы" на python </h3>
<p>Игра сделана с помощью библиотеки pygame на python. Фон в игре это зацикленная картинка(если использовать симетричную картинку, то стыка видно не будет), сам кот это несколько пнг картинок(чем больше картинок, тем плавнее передвижение) которые меняются с частотой 0.3 секунды.
Передвижение кота: переменная player_direction хранит состояние кота, т.е 0 - стоит, 1 - идет вправо, -1 - идет влево, для перемещения используется условие if-else, с помощью библиотеки отслеживаем нажатие кнопок на клавиатуре,
чтобы кот не убегал за пределы экрана используется условие, указываем растояние до кота справа и слева + при движении кота, двигаем экран на такое же расстояние. Для рыбы используем список и устанавливаем колличество рыб 0, fish_list = []
score = 0. С помощью цикла for добавляем рыбу в список, после того как кот относит рыбу в дом, выводим текст "+1" и удаляем картинку. Спавн рыбы происходит по таймеру (каждые 10 секунд), устанавливаем случайный диапазон для спавна fish_x = random.randint(0, 950)
(второй экран). В конце прописываем обновление экрана и выход из игры. </p>
